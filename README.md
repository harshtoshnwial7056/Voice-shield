# 🎤 VoiceShield – AI Voice Detection System

VoiceShield is a full-stack AI-powered web application designed to detect whether an audio input is generated by a human or an artificial intelligence system. With the rapid advancement of AI voice synthesis and deepfake technologies, VoiceShield aims to provide a reliable solution to identify and prevent voice-based fraud and impersonation.

---

## 🚀 Features

* 🎤 Detect whether a voice is **Human or AI-generated**
* ⚙️ Backend built with **Node.js & Express.js**
* 🧠 Integrated **Python Machine Learning Model** for audio classification
* 🌐 Frontend developed using **React.js**
* 🔐 Secure Authentication System (Register/Login)
* 🔑 Forgot Password functionality with email support
* ☁️ Cloudinary integration for audio file storage
* 📊 MongoDB database for storing users & prediction results

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/harshtoshnwial7056/Voice-shield.git
cd Voice-shield
```

---

### 2️⃣ Backend Setup

```
cd Backend
npm install
```

Create `.env` file:

```
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
CLOUDINARY_CLOUD_NAME=xxx
CLOUDINARY_API_KEY=xxx
CLOUDINARY_API_SECRET=xxx
```

Run backend:

```
node server.js
```

---

### 3️⃣ Frontend Setup

```
cd ../voiceshield-frontend
npm install
npm start
```

---

### 4️⃣ Python ML Setup

```
pip install -r requirements.txt
```

Run model manually (for testing):

```
python predict.py
```

---

## 🔄 Workflow

1. User uploads an audio file from the React frontend
2. File is sent to Express backend
3. Audio is uploaded to Cloudinary
4. Backend calls Python ML model
5. Model processes audio & predicts (AI/Human)
6. Result is stored in MongoDB
7. Response is sent back to frontend

---

## 🔐 Authentication Flow

* User registers with email & password
* Password is securely hashed using bcrypt
* JWT token is generated for authentication
* Forgot Password feature allows recovery via email
